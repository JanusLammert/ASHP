# ASHP
Auto-Selection-Helical-Polymers
Tool to do Autoselection of 2D classes based on the 2Dclass-averages.

# Prerequisits
1. Conda

# Installation
1. Install conda env
2. Download all Python skripts
3. Add folder to path
4. chmod

# Guide
1. conda activate ASHP
2. ASHP.py --help
3. conda deactivate
